## 수집 진행 중

### KB부동산 (kbland.kr)

| 항목 | 내용 |
|------|------|
| **적합성** | **확인 중** - 공개 API + CDP 기존 탭 방식으로 전환 |
| **상태** | **진행 중** - 공개 API는 단지/클러스터만 반환, 개별 매물은 인증 API 필요 |
| **수집 방식** | 1차) 공개 API (api.kbland.kr) 단지/클러스터 조회 → 2차) CDP 기존 kbland.kr 탭에서 인증 API 호출 |
| **수집기** | `scripts/kbland_auto_collector.mjs` |
| **테스트 결과** | 4개 구 테스트: 단지 5개 발견, 매물 0건 (공개 API만으로는 가격/면적 미포함, kbland.kr 로그인 탭 필요) |
| **다음 단계** | Chrome에서 kbland.kr 로그인 후 재테스트 (인증 API로 개별 매물 데이터 접근 시도) |
